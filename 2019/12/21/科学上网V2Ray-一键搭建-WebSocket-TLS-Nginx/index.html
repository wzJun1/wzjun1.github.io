科学上网V2Ray 一键搭建 + WebSocket + TLS + Nginx

发表于 2019-12-21 | 分类于 其他技术

前言

最近喵帕斯的服务有点不太稳定，卡顿严重，于是自己搞了一台谷歌云的服务器挂V2Ray来FQ，毕竟程序员的工作离不开Google

准备

  • 海外服务器一台
  • 域名一枚
  • V2Ray
  • 宝塔面板

安装

1.安装宝塔

请到 https://www.bt.cn/ 官网自己安装一键脚本

安装完成后记得安装Nginx 别装Apache

然后自己解析一个二级域名上来，添加一个网站

2.安装V2Ray

使用233的一键脚本安装

如果提示 curl: command not found，那是因为你的 VPS 没装 Curl
ubuntu/debian 安装 Curl:apt-get update -y && apt-get install curl -y
centos 安装 Curl: yum update -y && yum install curl -y

然后选择安装，即是输入 1 回车
选择传输协议，输入4 即选择 WebSocket + TLS

然后会出现 安装 Caddy 来实现 自动配置 TLS
选N 不安装Caddy 不自动配置 我们后面手动配置

是否配置 Shadowsocks ，如果不需要就直接回车，否则就输入 Y 回车

中间还要输入之前绑定的域名之类的，这里没有截图，自己看下

安装信息如果确保没有什么问题了，按回车继续

安装完成后 命令行输入 v2ray infolink 会给你一个V2Ray的客户端链接，保存

然后到宝塔里网站配置，添加一个反向代理，高级配置默认目录为/

目标URL为http://127.0.0.1:你的V2Ray端口 提交

然后在反向代理配置文件中，添加

如果看不懂，就看傻瓜式的说了一大堆的乱七八糟的文档系列：

https://233v2.com/post/17/

https://toutyrater.github.io/

https://liubing.me/v2ray-websocket-tl-nginx-cloudflare-bandwagonhost.html

完成

然后使用客户端连接之前保存的V2Ray的链接
